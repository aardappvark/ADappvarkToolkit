```

1. Install Shizuku from: https://github.com/RikkaApps/Shizuku/releases

3. Enable Wireless Debugging
4. Start Shizuku app and pair
5. Install ADappvark Toolkit
6. Grant Shizuku permission when prompted

TEST SCENARIOS:

1. HOME SCREEN
   - Verify setup checklist displays
   - Check Shizuku status indicators

2. SUBSCRIPTION (Use Devnet/Testnet)
   - View all three pricing plans
   - Tap "Subscribe Now"
   - Complete payment with test wallet
   - Verify subscription activates

3. UNINSTALL FEATURE
   - Navigate to Uninstall tab
   - Select 2-3 test apps (suggest installing test dApps first)
   - Tap "Uninstall Selected"
   - Confirm operation
   - Verify apps are removed

4. PERMISSIONS
   - Verify Shizuku permission request is clear
   - Check no excessive permissions requested

PAYMENT TESTING:
We recommend using devnet/testnet for review. The app supports both:
- Devnet: api.devnet.solana.com
- Mainnet: api.mainnet-beta.solana.com

Configuration can be changed in AppConfig.kt for testing.
```
