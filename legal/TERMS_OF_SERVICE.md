## 3. Description of Service

### 3.1 Core Features

ADappvark Toolkit is a mobile utility application that provides:

**Bulk Uninstall Feature**
- Scan and identify installed dApps from the Solana dApp Store
- Select multiple applications for batch uninstallation
- Silent uninstall via Shizuku integration (when available)
- Fallback to standard Android uninstall dialogs

**Bulk Reinstall Feature**
- View history of previously uninstalled applications
- Select applications for batch reinstallation
- Automated navigation to dApp Store for reinstall
- Accessibility service integration for streamlined workflow

**Storage Management**
- Display storage usage by installed applications
- Sort and filter applications by various criteria
- Track uninstall history for potential reinstallation

- Does NOT guarantee any airdrop eligibility or rewards
- Does NOT guarantee applications can be reinstalled after uninstallation

- May experience interruptions due to blockchain network conditions

### 3.3 Service Availability

We strive to maintain Service availability but do not guarantee:

- Uninterrupted access to the Service

- Compatibility with future Android or Seeker updates
